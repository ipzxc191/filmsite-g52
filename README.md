# FilmSite

Учебный проект по Django, который разрабатывается на протяжении всего курса.

В начале курса проект представляет собой небольшой сайт с каталогом фильмов. По мере изучения новых тем функциональность будет постепенно расширяться: появятся новые модели, административная панель, система пользователей, формы, загрузка файлов, аутентификация, оптимизация запросов и другие возможности Django.

---

## Установка проекта

### Способ 1. Скачать ZIP-архив (рекомендуется)

1. Откройте страницу репозитория.
2. Нажмите **Code → Download ZIP**.
3. Распакуйте архив в удобное место.
4. Откройте папку проекта в вашей IDE.

### Способ 2. Клонировать через Git

Если вы уже знакомы с Git, можете клонировать репозиторий командой:

```bash
git clone https://github.com/ipzxc191/filmsite-g52.git
```

Если вы еще не работали с Git, воспользуйтесь материалом курса:

[УРОК_ПО_GIT](https://github.com/ipzxc191/python-backend-course/blob/main/git-github-solo/outline.md)

## Запуск проекта

### 1. Создайте виртуальное окружение

```bash
python -m venv venv
```

### 2. Активируйте его

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустите сервер разработки

```bash
python manage.py runserver
```

После запуска откройте браузер и перейдите по адресу:

```
http://127.0.0.1:8000/
```

---

### 5. Заполнение данными

**Что внутри:**

- 11 жанров, 
- 15 режиссёров,
- ~38 актёров,
- 20 фильмов с реальными годами/описаниями/рейтингами.

**Запуск:**

```bash
python manage.py seed_films
```

Если нужно начать с чистой БД, есть флаг:

```bash
python manage.py seed_films --flush
```
